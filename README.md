# NewsAI
AI-powered global news summary app


## Team Members & Roles
- Student 1: [PengPai] | Project Leader / Frontend Development
- Student 2: [Hein] | Backend & Data Collection


## Project Goal
Provide users with daily one-sentence AI summaries of top news headlines from around the world.

## Initial Backlog Ideas
1. As a user, I want to view today’s top global news so I can quickly know world events.
2. As a user, I want each news to have a one-sentence AI summary so I don’t need to read long articles.
3. As a user, I want to refresh news so I can get the latest updates.
4. As a user, I want news categorized by topics (world, tech, sports, etc.) so I can read what I like.
5. As a user, I want to click a summary to view the full original news.
6. As a user, I want a simple and clean interface so I can use the app easily.
7. As a user, I want to browse news from different countries so I can understand global events.
